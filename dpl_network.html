Guided Technology Project: DPL Network Infrastructure

Abstract

This project focuses on the design and implementation of a new network infrastructure for Dublin Pharmaceutical Limited (DPL), aimed at adapting the company's technology to meet its growth and expansion needs. The project ensures security, productivity, and cutting-edge communication solutions using modern technology.

Project Overview

DPL, a pharmaceutical company, required an overhaul of its IT infrastructure due to recent expansions. The project covered networking, cloud computing, and security to ensure seamless communication between branches and business partners, enhancing productivity and operational efficiency.

Objective

The project aimed to build a resilient, high-performance network infrastructure capable of supporting DPL's future growth. It included high-availability network topology, virtualization, and security best practices, ensuring secure communication and business continuity.

Key Technologies Used

  • Cisco Hierarchical Network Design
  • Virtual LANs (VLANs) for network segmentation
  • Open Shortest Path First (OSPF) and Hot Standby Router Protocol (HSRP) for routing and fault tolerance
  • CentOS for hosting key services such as DHCP, DNS, and web services
  • Google Cloud Platform (GCP) for cloud-based backup and scalability

Implementation Steps

  1. Network Topology: Design of a high-availability topology, utilizing VLANs to segment DPL's departments (Manufacturing, R&D, Sales).
  2. VoIP Implementation: Setup of VoIP communication using Cisco 2811 routers for a modern, cost-efficient communication system.
  3. Security Best Practices: Use of RADIUS for centralized authentication and encryption mechanisms to secure device management.
  4. Cloud Integration: Setup of virtual machines on Google Cloud to ensure redundancy and scalability, supporting business continuity.

Testing and Evaluation

The project was rigorously tested for connectivity, security, and performance using a layered approach based on the OSI model. Virtualization services and cloud-based backups were also thoroughly tested to ensure redundancy.

Conclusion

The DPL network infrastructure project successfully established a secure, scalable, and cost-efficient network, ensuring long-term sustainability and business growth.
